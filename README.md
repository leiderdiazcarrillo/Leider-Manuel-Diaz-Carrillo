### 🛠️ Proyectos destacados

#### 📅 [EventPlanner SENA](https://github.com/tu-usuario/eventplanner-sena)
Sistema institucional de gestión de eventos para el SENA, construido en **ASP.NET MVC 5 (.NET Framework 4.8)** con **SQL Server** y **Bootstrap 5**, desplegado en **Azure App Service**.
`C#` `ASP.NET MVC` `SQL Server` `Azure` `Bootstrap`

#### 📚 [LibrosApp](https://github.com/tu-usuario/librosapp)
Aplicación **ASP.NET Core 8 MVC** para registro de libros, con escritura simultánea en **SQL Server** (Entity Framework) y **MongoDB** (MongoDB.Driver).
`ASP.NET Core` `Entity Framework` `MongoDB` `C#`

#### 🎉 [SENDAPLUS](https://github.com/tu-usuario/sendaplus)
Aplicación de escritorio en **C# (WinForms)** con **MongoDB** y **MaterialSkin**, para la gestión de eventos e invitados. Arquitectura en 3 capas con manual técnico completo.
`C#` `WinForms` `MongoDB` `MaterialSkin`

#### 🪪 [Tarjeta de Perfil](https://github.com/tu-usuario/tarjeta_perfil)
Tarjeta de perfil interactiva desarrollada en **Flutter**, con botones de redes sociales, chips de habilidades, sección de portafolio, certificaciones y función de compartir.
`Flutter` `Dart`
